Tentang Aplikasi Ini

Aplikasi web sederhana ini dibuat menggunakan Node.js dan Express.js sebagai pembelajaran dasar pengembangan web dengan JavaScript di sisi server.

Teknologi yang Digunakan:

  • Node.js - Runtime JavaScript untuk server-side
  • Express.js - Framework web minimalis untuk Node.js
  • HTML5 - Markup language untuk struktur halaman
  • CSS3 - Styling dan layout halaman
  • JavaScript - Interaktivitas di sisi client

Fitur yang Tersedia:

  • Server Express.js dengan routing
  • Serving static files (CSS, JS, images)
  • RESTful API endpoints
  • Form handling dengan POST request
  • Responsive design
  • Error handling (404 page)

Struktur Project:

├── index.js           # Main server file
├── package.json       # NPM configuration
├── views/            # HTML templates
│   ├── index.html
│   ├── about.html
│   └── 404.html
└── public/           # Static assets
    ├── css/
    │   └── style.css
    └── js/
        └── script.js
